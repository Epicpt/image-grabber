# Grabber concept arts "The book of Boba Fett" Season 1 from Starwars site.


Grabber concept arts - это функция на Go для скачивания 100+ изображений для своего рабочего стола с сайта https://www.starwars.com/the-book-of-boba-fett-season-1-concept-art-gallery .


## Установка

### Склонируйте репозиторий
```
git clone https://github.com/Epicpt/image-grabber.git
cd image-grabber
go mod tidy
```
## Запуск
```
go run main.go
```

Изображения скачиваются в папку image-grabber, откуда вы запускали исполняемый файл.
